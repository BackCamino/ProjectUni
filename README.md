# Project UNI
Progetto per esame universitario del corso metodologie di programmazione dell'università di Camerino.
Anno accademico 2025/2026

## 🤔​​ Che cosa è
ProjectUni è un alpha di un gioco rpg task manager che ambienta il giocatore in un università generica facendogli fare il ruolo dello studente. Il giocatore dovrà decidere tutte le attività che lo studente svolgerà. Le attività sono divise in task e simulano le attività comuni che ogni studente svolge: 
* andare a lezione,
* studiare
* dare gli esame.
* praticare sport
* E altro ancora.
  
## 🕹️​ Cosa si puo fare
Una volta creato il personaggio, il gioco stan nel pianificare le attività, la meccanica viene spiegata meglio nella [wiki](Wiki/Home.md)

## 🚀 Come eseguire il progetto

### Prerequisiti
- Java 25 (LTS)
- Gradle

### Istruzioni

```bash
git clone <https://github.com/BackCamino/ProjectUni.git>
```

### Build del progetto
```bash
./gradlew build
```

### Esecuzione
```bash
./gradlew run
```
